## Overview
Вариант 5. Проект получает информацию о людях. Далее выводит список с людьми с указанным месяцем рождения, предварительно отсортированные по своим инициалам
## Author
Зорин Даниил Олегович, группа 10.
## Usage
- (Для компиляции) Из корня проекта вызвать утилиту `make`.
- В папке `bin` запустить файл `main`.
- Далее следовать указаниям программы.
