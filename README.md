## Overview
Вариант 5. Проект получает информацию о рабочих. Далее выводит список с рабочими годом вступления в должность, равным 2020
## Author
Зорин Даниил Олегович, группа 10.
## Usage
- (Для компиляции) Из корня проекта вызвать утилиту `make`.
- В папке `bin` запустить файл `main`.
- Далее следовать указаниям программы.
